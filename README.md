# 🖼️ ComfyView Next-Gen
**"가장 빠르고 편안한 이미지 브라우징, 그리고 지능적인 프롬프트 압축 도구"**

ComfyView는 ComfyUI 및 Automatic1111과 같은 AI 이미지 생성 도구를 사용하는 유저들을 위해 설계된 고성능 이미지 브라우저이자 와일드카드 제작 도구입니다. 단순히 이미지를 보는 것을 넘어, 수천 장의 이미지 속에 숨겨진 프롬프트 데이터를 정제하고 압축하여 나만의 **무작위 프롬프트 자산(Wildcard)**을 만드는 데 최적화되어 있습니다.

---

## 💡 제작 의도 (Design Intent)

우리는 AI 이미지를 생성하면서 두 가지 문제에 직면합니다.
1. **정리의 늪**: 맘에 드는 이미지는 따로 보관하고, 실패작은 버려야 하는데 윈도우 기본 탐색기는 너무 느리고 번거롭습니다.
2. **프롬프트 과부하**: 좋은 프롬프트를 모으고 싶지만, 수천 개의 텍스트를 일일이 복사해서 붙여넣는 것은 불가능에 가깝습니다.

ComfyView는 **"최소한의 클릭으로 최대의 정리 효율"**을 내고, **"복잡한 프롬프트 세트를 한 줄의 지능적인 와일드카드로 압축"**하기 위해 태어났습니다.

## ✨ 핵심 기능 (Core Features)

### 1. 초고속 탐색 및 메타데이터 뷰어
- **병렬 PNG Chunk 파싱**: `Rayon` 라이브러리를 이용한 멀티스레딩 최적화로 이미지 메타데이터를 초고속으로 읽어옵니다.
- **SQLite 인덱싱**: 모든 폴더 데이터는 데이터베이스에 저장되어 수만 장의 이미지도 즉시 스크롤할 수 있습니다.
- **고급 재귀 검색 및 정렬**: 현재 폴더뿐만 아니라 모든 하위 폴더의 이미지를 동시에 검색할 수 있으며, 사용자가 설정한 정렬 방식(이름순, 날짜순 등)이 검색 결과에도 완벽하게 적용됩니다.

### 2. 효율적인 분류 및 관리 시스템
- **지능형 삭제 및 보관**: 파일을 `_Trash` 또는 `_Keep` 폴더로 빠르게 분류합니다.
- **실행 취소 (Undo)**: `Ctrl+Z`를 통해 실수로 옮긴 파일을 즉시 원래 위치로 되돌릴 수 있습니다. (최대 50개 히스토리)
- **배치 모드 (Batch Mode)**: 그리드 뷰에서 대량의 이미지를 한눈에 확인하고 관리합니다.

### 3. Wildcard Workshop (프롬프트 분석 도구)
- **유사도 기반 자동 병합**: 프롬프트 간의 Jaccard 유사도를 분석하여 지능적으로 와일드카드 패턴을 생성합니다. (기본 유사도 0.5)
- **유사도 순 결과 정렬**: 생성된 결과물들이 서로 비슷한 태그 구성을 가진 것끼리 모여있도록 정렬되어 분석이 용이합니다.
- **필터 병합 및 지속성**: 제외 필터 목록을 텍스트 입력이나 파일로 병합할 수 있으며, 모든 설정값은 세션이 종료되어도 안전하게 유지됩니다.

### 4. 개발자 도구 및 안정성
- **백엔드 디버거 (Ctrl+Alt+D)**: 인덱싱 상태, DB 경로, 샘플 데이터 등을 즉시 확인할 수 있는 내장 디버거를 지원합니다.
- **안정적인 데이터 관리**: 모든 설정과 DB는 시스템의 전용 앱 데이터 디렉토리에 저장되어 권한 문제나 파일 유실을 방지합니다.

## 🚀 워크플로우 가이드 (Recommended Workflow)

프로그램을 처음 사용하는 분들을 위한 가장 효율적인 작업 흐름입니다.

### 1단계: 탐색 및 분류 (Browsing & Sorting)
1. 이미지가 가득한 폴더를 드래그 앤 드랍하여 엽니다.
2. 헤더의 **정렬 아이콘**을 눌러 원하는 순서로 배치합니다. (재귀 모드 지원)
3. 이미지를 하나씩 보며(또는 배치 모드에서) 맘에 드는 건 `K`(Keep), 버릴 건 `Delete`(Trash)를 눌러 정리합니다. 실수했다면 `Ctrl+Z`로 되돌립니다.

### 2단계: 와일드카드 소스 수집 (Importing)
1. 상단 헤더의 **Wildcard** 버튼을 눌러 워크숍을 엽니다.
2. **Import from Viewer**를 눌러 방금 고른 이미지들을 워크숍으로 가져옵니다. 

### 3단계: 압축 및 생성 (Compression)
1. **Similarity(유사도)** 슬라이더를 조절합니다. (0.5 권장)
2. **Generate** 버튼을 누르면 병렬 처리로 와일드카드가 생성됩니다.
3. 결과물은 서로 비슷한 것끼리 묶여 정렬됩니다. **Copy**하거나 **Export .txt**로 내보내세요!

## ⌨️ 단축키 (Shortcut Keys)

- `→` / `←` : 다음/이전 이미지
- `Delete` : 쓰레기통으로 이동 (`_Trash` 폴더)
- `K` : 보관함으로 이동 (`_Keep` 폴더)
- `Ctrl + Z` : 분류 동작 되돌리기 (Undo)
- `B` : 배치 모드 (그리드 보기) 토글
- `/` : 프롬프트 검색창 포커스
- `R` : 새로고침 (이미지 재인덱싱)
- `Ctrl + Alt + D` : 백엔드 디버그 패널 토글

## 🛠️ 실행 방법 (How to Run)

Windows 환경에서는 루트 폴더의 `run.bat` 파일을 실행하는 것이 가장 간편합니다.

1. `run.bat` 실행 (자동으로 의존성을 설치하고 개발 서버를 시작합니다).
2. 최초 실행 시 Rust 컴파일로 인해 시간이 약간 소요될 수 있습니다.
