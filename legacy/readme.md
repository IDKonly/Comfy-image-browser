# Image Manager (Optimized)

**AI 생성 이미지 관리를 위한 고성능 데스크톱 애플리케이션**

이 프로그램은 `Stable Diffusion`, `ComfyUI` 등으로 생성된 대량의 이미지를 효율적으로 탐색하고 관리하기 위해 제작되었습니다. 특히 PNG 메타데이터(Prompt, LoRA 등)를 자동으로 색인화하여 강력한 검색 기능을 제공합니다.

---

## 🚀 주요 기능 (v2.0)

### 1. ⚡ 압도적인 성능
*   **비동기 이미지 로딩:** 고해상도 이미지를 볼 때 UI가 멈추지 않습니다.
*   **스마트 프리페칭(Pre-fetching):** 사용자가 볼 다음 이미지를 백그라운드에서 미리 준비하여 즉시 보여줍니다.
*   **DB 기반 색인:** `SQLite`를 사용하여 수만 장의 이미지도 파일 수정 시간(`mtime`) 기반으로 **변경된 파일만** 빠르게 로딩합니다.
*   **최적화된 파싱:** C++ 수준의 정규식 컴파일과 문자열 처리를 통해 메타데이터 파싱 속도를 32% 향상시켰습니다.

### 2. 🔍 강력한 관리 도구
*   **메타데이터 검색:** 프롬프트, 네거티브 프롬프트, LoRA 이름으로 이미지를 즉시 검색합니다.
*   **워크플로우 관리:** 삭제(`Delete`), 보관(`_Keep` 폴더 이동), 회전, 줌 등 필수 기능을 단축키로 제공합니다.
*   **드래그 앤 드롭:** 폴더나 이미지를 앱에 드래그하여 바로 엽니다.

### 3. 🎨 모던 UI
*   `CustomTkinter` 기반의 깔끔한 다크/라이트 모드 지원.
*   반응형 레이아웃 및 썸네일 미리보기 바.

## 🛠 설치 및 실행

### 1. 필수 요구 사항
*   Windows 10/11
*   Python 3.10 이상

### 2. 설치
터미널에서 필수 라이브러리를 설치합니다.

```bash
pip install customtkinter tkinterdnd2 pillow
```

### 3. 실행
*   **방법 A:** `run.bat` 파일을 더블 클릭합니다. (추천)
*   **방법 B:** 터미널에서 `python main.py`를 입력합니다.

## 🎮 단축키 가이드

| 기능 | 단축키 | 설명 |
| :--- | :--- | :--- |
| **이전 이미지** | `←` (왼쪽 화살표) | 이전 이미지로 이동 |
| **다음 이미지** | `→` (오른쪽 화살표) | 다음 이미지로 이동 |
| **삭제** | `Delete` | 현재 이미지를 `_Trash` 폴더로 이동 |
| **보관** | `K` | 현재 이미지를 `_Keep` 폴더로 이동 |
| **회전** | `L`, `R` | 왼쪽/오른쪽으로 90도 회전 |
| **줌** | `+`, `-` | 이미지 확대/축소 |
| **보기 전환** | `V` | 화면 맞춤 / 원본 비율 전환 |
| **폴더 열기** | `Ctrl + O` | 탐색기로 폴더 선택 |
| **새로고침** | `F5` | 파일 목록 및 DB 동기화 |

## 📂 프로젝트 구조 (MVC 패턴)

```
ImageManager/
├── main.py             # 프로그램 진입점
├── app_controller.py   # [Controller] 로직 및 이벤트 제어
├── ui_view.py          # [View] UI 구성 및 위젯 관리
├── db_manager.py       # [Model] SQLite DB 및 검색 엔진
├── image_loader.py     # [Model] 비동기 이미지 로더 스레드
├── image_utils.py      # [Util] 고성능 메타데이터 파서
├── config.py           # 설정 관리
└── run.bat             # 윈도우 실행 스크립트
```
