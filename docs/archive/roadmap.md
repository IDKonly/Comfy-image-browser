# 프로젝트 로드맵: ComfyView (Project Roadmap)

이 로드맵은 ComfyView의 주요 개발 마일스톤과 향후 계획을 나타냅니다.

## Phase 1: 기반 구축 및 고성능 코어 (완료)
- [x] 기술 스택 전환: Tauri v2 + React + Rust.
- [x] 고속 디렉토리 스캐닝 (Rust 기반).
- [x] PNG 메타데이터 (tEXt/iTXt) 추출 및 파싱.
- [x] 기본 파일 작업 (삭제, 유지).
- [x] 키보드 네비게이션.

## Phase 2: 시각적 피드백 및 GUI/UX 고도화 (완료)
- [x] **시각적 피드백 시스템**:
    - [x] 파일 작업 시 Toast 알림 추가.
    - [x] 이미지 간 부드러운 전환 효과.
- [x] **고급 뷰어 UX**:
    - [x] 고성능 Zoom/Pan 구현.
- [x] **레이아웃 최적화**:
    - [x] 그리드 뷰 (Batch Mode) 지원.
    - [x] 현대적인 다크 테마 적용.

## Phase 3: 시나리오 중심의 기능 통합 및 지능화 (완료)
- [x] **배치 로직**:
    - [x] 인접 프롬프트 그룹화 로직 포팅.
    - [x] 배치 모드에서 시각적 그룹화 인디케이터 제공.
- [x] **고급 검색 및 필터**:
    - [x] 다중 태그 검색 (AND 조건).
    - [x] 모델/샘플러별 필터링.
    - [x] 재귀적 폴더 스캔 및 검색 지원.
- [x] **와일드카드 워크숍**:
    - [x] AI 기반 프롬프트 압축 기술 적용.
    - [x] 유사도 기반 결과 정렬.
    - [x] 대화형 태그 리파이너(Refiner) 추가.

## Phase 4: 성능 및 안정성 강화 (완료)
- [x] **가상화 리스트**: 10,000장 이상의 이미지를 위한 고성능 가상 스크롤 지원.
- [x] **멀티스레딩 최적화**: 병렬 메타데이터 인덱싱 및 썸네일 생성.
- [x] **실행 취소(Undo) 시스템**: Ctrl+Z로 우발적인 파일 이동 복구.
- [x] **데이터 보존**: 앱 데이터 디렉토리에 설정 및 데이터베이스 안전하게 저장.

## Phase 5: 고급 지능화 및 생태계 확장 (진행 중)
- [x] **단순 와일드카드 모드**: 재귀적 압축을 건너뛰고 순수 유니크 태그만 추출하는 옵션 추가.
- [x] **소셜 공유**: 클릭 한 번으로 트위터(X) 공유 (태그 자동 추출 및 클립보드 연동).
- [x] **ComfyUI 최신 노드 확장**: SDXL, SD3, UNETLoader 등의 메타데이터 정확한 추출 지원.
- [ ] **AI 자동 태깅**: 로컬 LLM/Vision 모델을 사용하여 메타데이터가 없는 이미지의 태그 생성.
- [ ] **중복 이미지 감지**: 내용 기반으로 완전히 동일하거나 유사한 이미지 찾기 및 그룹화.
- [ ] **모바일 리모트**: 동일 네트워크 내의 모바일 기기에서 이미지 브라우징 및 분류 지원.
- [ ] **확장 API**: 커뮤니티 제작 테마 및 플러그인 지원.
