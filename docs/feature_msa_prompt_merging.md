# MSA 기반 지능형 프롬프트 병합 알고리즘 (MSA-based Intelligent Prompt Merging)

이 문서는 MSA(Multiple Sequence Alignment) 알고리즘을 프롬프트/태그 병합에 도입하여, 데이터의 손실 없이 최적화된 와일드카드 템플릿을 생성하는 지능형 알고리즘 설계안을 다룹니다.

## 1. 핵심 개념 (The Core Idea)

생물학에서 DNA나 단백질 서열의 유사성을 찾기 위해 사용하는 **MSA**를 프롬프트 분석에 도입합니다. 여러 개의 프롬프트를 정렬(Alignment)하여 공통 분모와 가변 분모(Gap)를 식별하고, 이를 통해 `{A|B|C}` 형태의 와일드카드 슬롯을 자동으로 생성합니다.

*   **원본 데이터**: `A, C, E` / `A, B, C, D, E`
*   **MSA 정렬 결과**: `A, -, C, -, E` / `A, B, C, D, E`
*   **병합 템플릿**: `A, {B|}, C, {D|}, E`

---

## 2. 도전 과제 (Challenges)

프롬프트는 일반적인 서열(Sequence) 데이터와 다른 두 가지 결정적인 특징이 있으며, 이를 해결해야만 실용적인 병합이 가능합니다.

1.  **순서 무관성 (Order Independence)**: 태그의 나열 순서가 결과에 미치는 영향이 미미하거나 무작위적입니다.
2.  **의미론적 표류 (Semantic Drift)**: 단순히 빈 공간을 채우는 방식은 `underwater`와 `fire`가 섞이는 것과 같은 비현실적인 태그 조합을 생성할 확률이 높습니다.

---

## 3. 보완 전략 (Proposed Strategies)

### 3.1. 정규화된 순서 적용 (Canonical Ordering)
MSA 적용 전, 모든 프롬프트의 태그를 **알파벳순** 또는 **전체 DB 내 출현 빈도순**으로 정규화합니다. 이를 통해 순서가 무작위인 집합 데이터를 서열 데이터로 변환하여 MSA 알고리즘이 '의미상 동일한 위치'를 정확히 맵핑할 수 있도록 합니다.

### 3.2. 통계적 공생 빈도 검증 (Co-occurrence Filtering)
이상한 조합을 방지하기 위해 ComfyView의 SQLite DB를 활용한 통계 검증을 수행합니다.
*   **로직**: 생성된 와일드카드 슬롯 `{B|X}` 내의 각 태그가 주변의 확정 태그 `A, C, E`와 함께 실제로 사용된 적이 있는지 **상호 정보량(PMI)**을 체크합니다.
*   **효과**: 통계적으로 유효하지 않은 조합은 슬롯에서 배제하거나 사용자에게 경고를 표시합니다.

### 3.3. 조건부 동기화 슬롯 (Synchronized Slots)
단일 태그 단위가 아닌, **함께 움직여야 하는 태그 묶음**을 보존합니다.
*   **예시**: MSA 분석 결과 `B1`이 출현할 때 항상 `D1`이 함께 나타난다면, 이를 `{B1, D1 | B2, D2}`와 같은 결합 슬롯으로 병합합니다.
*   **효과**: "낮 배경" 태그와 "밤 조명" 태그가 섞이는 참사를 방지하고 컨셉의 일관성을 유지합니다.

### 3.4. 카테고리 기반 상호 배제 (Mutual Exclusion)
의미론적으로 양립할 수 없는 태그 그룹(예: `1girl` vs `multiple girls`)을 미리 정의하여, 하나의 병합 프롬프트 내에서 충돌하는 태그들이 동시에 활성화되지 않도록 제약 조건을 부여합니다.

---

## 4. 구현 로직 예시 (Implementation Path)

1.  **Preprocessing**: 입력된 프롬프트셋의 태그를 정렬 및 중복 제거.
2.  **Alignment**: 정렬된 서열을 바탕으로 가중치 기반 MSA 수행 (Gap Penalty는 태그 간 상관관계에 반비례).
3.  **Slot Synthesis**: 정렬된 결과에서 공백(Gap)이 발생한 지점을 와일드카드 슬롯으로 변환.
4.  **Refinement**: DB 쿼리를 통해 슬롯 내 태그와 주변 태그 간의 결합 적합도 검증.
5.  **Output**: 최적화된 `__wildcard__` 템플릿 출력.

---

## 5. 결론 및 기대 효과

이 방식은 단순히 태그를 합치는 것을 넘어, **데이터에 기반한 창작 의도(Context)를 보존하는 병합**을 지향합니다. 크리에이터는 수천 장의 이미지에서 자신도 모르게 반복하던 유효한 프롬프트 패턴을 MSA를 통해 '가장 정제된 형태의 템플릿'으로 추출할 수 있게 됩니다.
